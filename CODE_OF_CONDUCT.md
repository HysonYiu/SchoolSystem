# 貢獻者公約

## 我們的承諾

作為貢獻者和維護者，我們承諾在我們的社區中提供一個安全、包容的環境。我們致力於為所有人提供無騷擾的體驗，不論年齡、體型、外表、種族、民族、性別認同和表達、經驗水平、國籍、外表、身體能力、身體殘疾、宗教信仰或性認同和取向。

## 我們的標準

示範出以下行為的例子有助於創造積極的環境：

- 使用熱烈和包容的語言
- 尊重不同的觀點和經驗
- 優雅地接受建設性的批評
- 關注對社區最好的事情
- 表現對其他社區成員的同情

不可接受的行為例子包括：

- 涉及性內容或圖像的騷擾和暴力威脅
- 嘲弄、貶低性評論或人身攻擊
- 公開或私下騷擾
- 未經明確許可發佈他人的私人信息（例如，物理或電子地址）
- 其他不合理的行為，在專業環境中合理地被認為是不適當的

## 執行

社區領導人負責澄清和執行我們對可接受行為的標準，並將採取適當的公平糾正措施以應對他們認為是不適當、威脅、冒犯或有害的任何行為。

## 報告違規

具有關於不可接受行為的疑慮的社區成員可以聯繫項目維護者以進行報告。所有投訴將被審查和調查，並將導致被認為必要和適當的回應。項目團隊有義務對事件的記者保持機密。

## 執行指南

社區領導將遵循以下社區影響執行指南：

### 1. 更正

**社區影響**：不專業的語言或社區標準的其他行為。

**後果**：項目維護者的私人書面警告，明確說明違規的性質以及為什麼該行為不適當。可能要求公開道歉。

### 2. 警告

**社區影響**：違反社區標準的單一或一系列行為。

**後果**：具有持續不當行為的後果的警告。在指定時間內與涉及人員沒有互動，包括主動與執行代碼進行互動的非請求互動。這包括避免在社區空間（如討論、社交媒體）以及外部頻道中的互動。違反這些條款可能導致臨時或永久禁止。

### 3. 臨時禁止

**社區影響**：嚴重違反社區標準，包括持續不當行為。

**後果**：臨時禁止任何社區空間中的任何類型的互動或溝通一段時間。在此期間內不允許公開或私下與項目互動，包括與他人的互動。違反這些條款可能導致永久禁止。

### 4. 永久禁止

**社區影響**：展示了系統性違反社區標準的模式，包括持續不當行為、對個人的騷擾或對班級個人的侵略或貶低。

**後果**：永久禁止任何社區空間中的任何公開互動。
